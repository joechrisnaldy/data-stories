# Data

The data is not committed here. Download it from Kaggle.

## Source

[Financial Risk for Loan Approval](https://www.kaggle.com/datasets/lorenzozoppelletto/financial-risk-for-loan-approval)
(Zoppelletto, 2024), CC0: Public Domain. Last updated 2024-09-07.

This is a SYNTHETIC dataset: 20,000 fictional loan applicants, generated by a script, not real
people. Its own description opens "Synthetic Dataset for Risk Assessment and Loan Approval
Modeling." Nothing here is evidence about any real bank or any real borrower.

## Download

```bash
kaggle datasets download -d lorenzozoppelletto/financial-risk-for-loan-approval -p .
unzip financial-risk-for-loan-approval.zip
```

## The two files

- `Loan.csv`: 20,000 rows, 36 columns. Demographics, credit history, income, debt, and two
  targets: `LoanApproved` (approved/denied) and `RiskScore` (a continuous risk number).
- `CSV_Generation.py`: the generator script that BUILT the data. This is the unusual part. Because
  the dataset ships its own generator, we can read the exact rulebook: the weighted formula that
  decides `LoanApproved`, how the interest rate is set from the credit score, and how `RiskScore`
  is assembled. The essay reads this rulebook and then checks it against the 20,000 rows.

## What the generator encodes (read directly from `CSV_Generation.py`)

- Approval is a weighted score of income, debt-to-income, interest rate, loan amount, credit
  score, bankruptcy, past defaults, employment, home ownership, payment history, credit-history
  length, net worth, age, experience, education, and the application month, plus random noise. An
  applicant is approved when that score clears a threshold.
- The interest rate is mostly a function of the credit score (a lower score buys a higher rate),
  and it is the rate, not the score itself, that carries weight in the approval score.
- `RiskScore` is built by different logic (correctly-signed risk buckets), so the two targets only
  partly agree.

## Note

Every "rule" described in the essay is a fact about this synthetic generator, NOT a claim about how
any real lender behaves. All analysis here is descriptive and associational. Not financial advice.
